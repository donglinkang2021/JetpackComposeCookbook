# 快捷键

### 快速选中多个相同代码段

全选中代码段之后狂按<kbd>Alt</kbd>+<kbd>J</kbd>



### surround with widget

![image-20230318210200264](快捷键.assets/image-20230318210200264.png)

选中一行之后按下<kbd>crtl</kbd>+<kbd>alt</kbd>+<kbd>J</kbd>就可以弹出右边窗口，选中之后即可实现包裹功能；
